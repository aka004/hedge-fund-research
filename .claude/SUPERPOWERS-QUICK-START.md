# Superpowers Quick Start

## Installation (Run in Cursor Chat)

```bash
# Step 1: Register marketplace
/plugin marketplace add obra/superpowers-marketplace

# Step 2: Install plugin
/plugin install superpowers@superpowers-marketplace

# Step 3: Verify
/help
```

## Available Commands

After installation, these commands will be available:

- `/superpowers:brainstorm` - Interactive design refinement
- `/superpowers:write-plan` - Create implementation plan  
- `/superpowers:execute-plan` - Execute plan in batches

## Skills Auto-Trigger

These skills activate automatically (already referenced in CLAUDE.md):

- `superpowers:test-driven-development` - TDD workflow
- `superpowers:systematic-debugging` - Debug methodology
- `superpowers:requesting-code-review` - Code review
