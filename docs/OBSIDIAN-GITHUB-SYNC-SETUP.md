# Obsidian GitHub Sync Setup

## Overview

The [Obsidian GitHub Sync plugin](https://github.com/kevinmkchin/Obsidian-GitHub-Sync) allows you to sync your Obsidian vault to a GitHub repository, providing cross-device sync without iCloud Drive.

## Why Use GitHub Sync?

✅ **Reliable**: Works across all platforms (Mac, Windows, Linux, Mobile)  
✅ **Version Control**: Full git history of your notes  
✅ **No Permission Issues**: Direct git operations  
✅ **Backup**: Automatic backup to GitHub  
✅ **Free**: GitHub private repos are free

## Setup Instructions

### Step 1: Install the Plugin

1. Open Obsidian app
2. Go to **Settings** → **Community plugins**
3. Click **Browse** and search for "GitHub Sync"
4. Install the plugin by **kevinmkchin**
5. Enable the plugin

### Step 2: Initialize Git Repository

If `obsidian-vault` is not already a git repository:

```bash
cd /Users/yung004/Documents/claude_code/claude-code-workspace/hedge-fund-research/obsidian-vault
git init
git add .
git commit -m "Initial commit: Hedge Fund Research notes"
```

### Step 3: Create GitHub Repository

1. Go to [GitHub](https://github.com) and create a new repository
2. Name it: `hedge-fund-research-notes` (or your preferred name)
3. Choose **Private** (recommended for research notes)
4. **Don't** initialize with README (we already have files)

### Step 4: Connect to GitHub

**Option A: Using HTTPS (Easier)**
```bash
cd /Users/yung004/Documents/claude_code/claude-code-workspace/hedge-fund-research/obsidian-vault
git remote add origin https://github.com/YOUR_USERNAME/hedge-fund-research-notes.git
git branch -M main
git push -u origin main
```

**Option B: Using SSH**
```bash
git remote add origin git@github.com:YOUR_USERNAME/hedge-fund-research-notes.git
git branch -M main
git push -u origin main
```

### Step 5: Configure Plugin in Obsidian

1. Open Obsidian settings
2. Go to **GitHub Sync** settings
3. Enter your **Remote URL**:
   - HTTPS: `https://github.com/YOUR_USERNAME/hedge-fund-research-notes.git`
   - SSH: `git@github.com:YOUR_USERNAME/hedge-fund-research-notes.git`
4. (Optional) Set custom git path if needed
5. Save settings

### Step 6: Test Sync

1. Click the **Sync with Remote** button in Obsidian (ribbon icon)
2. Or use Command Palette: "Sync with Remote"
3. First sync will push your local files to GitHub

## Usage

### Manual Sync

- Click the **Sync with Remote** ribbon icon
- Or use Command Palette: "Sync with Remote"
- Plugin will pull changes, then push local changes

### Automatic Sync on Startup

Enable in plugin settings:
- **Auto-sync on startup**: Automatically syncs if behind remote

## Troubleshooting

### Authentication Issues

If prompted for credentials:
- **HTTPS**: Use Personal Access Token (not password)
- **SSH**: Ensure SSH key is set up in GitHub

### Conflicts

If conflicts occur:
- Plugin will open conflicted files
- Resolve manually or accept one version
- Sync again

### Git Not Found

If git is not in PATH:
- Set custom git path in plugin settings
- Example: `/usr/local/bin/git` (macOS)

## Benefits Over iCloud Drive

| Feature | iCloud Drive | GitHub Sync |
|---------|--------------|-------------|
| Permission Issues | Yes (restrictive) | No (direct git) |
| Version Control | No | Yes (full history) |
| Cross-Platform | Apple devices | All platforms |
| Backup | iCloud only | GitHub + local |
| Conflict Resolution | Manual | Git merge |
| Mobile Access | Obsidian app | GitHub app + Obsidian |
